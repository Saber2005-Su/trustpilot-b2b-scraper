# Trustpilot B2B Lead Scraper

## What it does
Scrapes business contact information from Trustpilot including name, phone, email, and website.

## Data collected
- Business name
- Phone number
- Email address
- Website
- City

## Tech stack
Python, Playwright

## How to run
pip install playwright
python scraper.py

## Output
Saves data to business_data.csv
